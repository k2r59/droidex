1. Principes de conception

Interface sombre, contrastée et immersive, mais avec des surfaces suffisamment différenciées pour rester lisible.

Un seul accent principal : le cyan. Le violet est réservé aux Nova Crystals et aux éléments extraordinaires.

Hiérarchie forte : titre de page, résumé, action principale, puis contenu détaillé.

Les informations de progression et de coût doivent être visibles sans ouvrir de détail.

Les illustrations servent d'ambiance et de repère, jamais de support unique à une information importante.

Les effets lumineux sont subtils : glow présent sur l'état actif, pas sur toutes les cartes.
Le style est inspiré d'une interface de jeu de science-fiction, mais les composants restent suffisamment sobres pour 
une utilisation documentaire et communautaire.

18. Accessibilité

Contraste minimum 4,5:1 pour les textes courants.

Ne jamais communiquer une rareté uniquement par la couleur : conserver un libellé textuel.

Ajouter un focus visible sur tous les contrôles clavier.

Les images décoratives utilisent alt="" ; les images porteuses d'information ont un alt descriptif.
